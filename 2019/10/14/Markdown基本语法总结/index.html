2019-10-14
  • IT

概述
Markdown是一种纯文本的轻量级标记语言，可简洁方便的生成结构化html文档。支持mac,windows,linux等操作系统，支持导出HTML、PDF、Word、图片等多种类型文件。

语法

1.标题
语法：使用 # 号可表示 1-6 级标题，一级标题对应一个 # 号，二级标题对应两个 # 号，以此类推。

对齐方式

我们可以设置表格的对齐方式：

-: 设置内容和标题栏居右对齐。

:- 设置内容和标题栏居左对齐。
